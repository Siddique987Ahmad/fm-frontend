üì¶ Products Collection Queries

View all products:

db.products.find().pretty()

View products by type:

db.products.find({productType: "white-oil"}).pretty()

View sales only:

db.products.find({transactionType: "sale"}).pretty()

View pending payments:

db.products.find({paymentStatus: "pending"}).pretty()

Products with amount > 100000:

db.products.find({totalBalance: {$gt: 100000}}).pretty()

Latest 5 products:

db.products.find().sort({createdAt: -1}).limit(5).pretty()

üí∞ Expenses Collection Queries

View all expenses:

db.expenses.find().pretty()

View expenses by category:

db.expenses.find({expenseCategory: "home"}).pretty()

View pending expenses:

db.expenses.find({paymentStatus: "pending"}).pretty()

Expenses with amount > 10000:

db.expenses.find({amount: {$gt: 10000}}).pretty()

Latest 5 expenses:

db.expenses.find().sort({createdAt: -1}).limit(5).pretty()

üîß Useful Commands

Delete a specific document:

db.products.deleteOne({_id: ObjectId("YOUR_DOCUMENT_ID")})

Update a document:

db.products.updateOne(
  {_id: ObjectId("YOUR_DOCUMENT_ID")}, 
  {$set: {status: "completed"}}
)

Create an index for better performance:

db.products.createIndex({productType: 1, createdAt: -1})

Drop entire collection (be careful!):

db.products.drop()
